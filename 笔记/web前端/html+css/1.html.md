### HTML

#### 乱码问题：

常见字符集：ASCII、ISO-8859-1、GBK、GB2312、UTF-8

```html
<!--meta标签用来设置网页的一些元数据，比如网页的字符集，关键字、简介 -->
<meta charset="utf-8"/>
<!-- 
	使用meta标签还可以用来设置网页的关键字
-->
<meta name="keywords" content="HTML5"/>

<!-- 
	还可以指定网页的描述
-->
<meta name="description" content="发布h5、等相关信息" />

<!-- 
	使用meta可以用来做请求的重定向
-->
<meta http-equiv="refresh" content="5";url=" " />
```



#### 常用的标签：

##### 标题标签：

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
<!--在显示效果上h1最大，h1非常重要，重要性仅次于title，页面只能写一个h1 ，一般只是用到三级标签-->


<!-- 段落标签
p标签中的文字，默认会独占一行，并且段于段会有一个间距
-->
<!-- 在html中字符之间有再多的空格，浏览器也会当作一个空格解析，换行也会当作一个空格解析-->
<p></p>

<!-- 在页面中可以使用br标签来表示一个换行，br标签是一个自结束标签-->
<br />
<!-- 在HTML当中，一些如< >这种特殊字符不能直接使用，需要使用一些特殊的符号表示这些特殊字符，这些特殊字符我们称为实体


实体的名字：

	< &lt 小于号
	> &gt 大于号
	空格 &nbsp;
	版权符号 &copy;

-->

<!-- 图片标签
属性：
	src：设置一个外部图片的路径
	alt：可以设置一个图片的描述(图片不能显示的时候)
		搜索引擎可以通过alt属性来识别不同的图片
		如果不写，则搜索引擎不会对其收录。
	width：可以修改图片宽度，一般用px作为单位。
	height：可以修改图片的高度，一般用px作为单位。
	
	宽度和高度两个如果值设置一个，另一个也会等比例调整大小。
	一般开发除了自适应页面，不建议使用weight和height
-->
<img src=" " alt="" />
<!-- 
	图片格式：JPEG（JPG）
				-支持的颜色比较多，图片可以压缩，但是不支持透明
				-一般使用来保存照片等
、			GIF、
				-支持的颜色少，只支持简单透明，支持动态图
				-图片颜色单一或者动态图时使用
			png
				-支持的颜色多，并且支持复杂的透明
				-可以用来显示颜色复杂的透明图片
-->


<!--xHtml语法规范
	1. html中不区分大小写，一般都是用小写
	2. html中的注释不能嵌套
	3. html标签必须结构完整，要么成对出现，要么自结束标签
	4.标签可以嵌套，但是不能交叉嵌套。
	5.属性必须有值，且值必须加引号。
-->

<!-- 内联框架
	使用iframe来创建一个内联框架
属性：
	src：指向一个外部页面的路径，可以使用相对路径。
	width：
	heigh：
	name：可以为内联框架设置一个名字
在现实开发中不推荐使用，因为内联框架中的内容不会被搜索引擎所检索。
	-->
<iframe src="路径"></iframe>
<!-- 超链接
		使用超链接可以让我们从一个页面跳转到另一个页面
		使用a标签来创建一个超链接
	属性：
		href：指向跳转的目标地址
		target:指定连接打开位置
				_self,默认
				_blank，在新窗口打开来链接
				可以设置一个内联框架name属性，在内联框架中打开
	如果将链接路径设置为#,则会默认回到顶部。
		
-->
<a href=” 路径“></a>
<a href="mailto:邮件地址"></a>
<!-- center中的内容默认在页面中居中显示-->
<center></center>
```
